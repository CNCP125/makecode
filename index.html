SENSEI GUIDANCE

BEFORE THE SESSION

  • An internet connection is required for the Game Building Session activities.
  • Have the Game Building Session landing page open and ready on screen for the child.
  • Check all links to games before the GBS begins by clicking the on-screen game options.

DURING THE SESSION

  • The GBS tutorial should take 15-20 minutes of the scheduled tour.
  • Encourage the child to read through each step on the left, drag out the corresponding code blocks in the editor, then click on the emulator icon (top left) or the mini-emulator (bottom right) to see how the new code affected the project.
  • The final step of each tutorial offers ideas for extending the project, if time permits.
  • Clicking “Exit Tutorial” at the top left of the screen will return to the MakeCode Arcade homepage. There, the child can click on their project to return to the tutorial, as needed. 

AT THE END OF THE SESSION

  • At the end of the tutorial, click “Done” in order to publish the project. Doing so will result in a link and a QR code that can be shared with parents/guardians and used to access the project at home after the tour.
  • Encourage the child to play the game multiple times and celebrate their accomplishment! If possible, invite the parents/guardians to play the game, too!
